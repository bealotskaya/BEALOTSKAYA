Семантические и функциональные строчные теги

Тег <STRONG> для выделения фрагмента текста более важным. Тэг <em> менее важным. В то время как теги <b> и <i> делают текст просто полужирным и курсивом соответственно.

Теги для обозначения машинного языка

Пример программы на языке PERL:

#!/usr/local/bin/perl
print "Content-type: text/html\n\n";
print "Hello, world!\n";

Программа выведет:

Hello, World!

Однако в браузере у нас будет отображаться результат выполнения этой программы:

Hello, World!
